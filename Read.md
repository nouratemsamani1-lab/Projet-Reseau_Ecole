# TP : Page Web HTML + CSS

## Choix de design
J'ai choisi une palette simple avec du bleu (#4B7BE5) et un accent orange (#F2A541), pour donner une identité visuelle douce et moderne.

## Mise en page
- Flexbox utilisé pour mettre le tableau et le formulaire en deux colonnes.
- Responsive : en dessous de 700px, la page repasse en une colonne.

## Difficultés rencontrées
- Organisation du formulaire avec fieldset/legend.
- Styliser les tableaux avec effet hover et alternance de lignes.

## Accessibilité
- Labels associés à chaque input.
- Focus visible sur les champs du formulaire.
